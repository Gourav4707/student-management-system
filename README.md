Student Management System

A simple command-line based Student Management System built using Python. This project demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, methods, and encapsulation.

✨ Features

➕ Add a new student

📋 Display all students

🔍 Search student by ID

✏️ Update student details

❌ Delete a student

🛡️ Prevent duplicate student IDs

🛠️ Technologies Used
Python 3

Object-Oriented Programming (OOP)

Command Line Interface (CLI)

📂 Project Structure
🚀 How to Run
1. Clone the repository
2. Run the program

Make sure Python 3 is installed on your system.
